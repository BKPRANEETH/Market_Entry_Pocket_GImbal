# Creator Economy Gear — Market Entry Case Study

A beginner-level end-to-end market research and market entry 
case study for a creator economy gear brand targeting Indian 
content creators.

## What This Project Covers
- Industry overview and market sizing
- Target audience and buyer persona development
- Competitive analysis and pricing strategy
- Financial modelling (break-even, margins, ROI, projections)
- Go-to-market and marketing strategy
- SQL analysis and Excel financial model
- Data visualisation dashboard

## Tools Used
- Microsoft Excel — financial modelling and dashboards
- SQL — data querying and customer segmentation
- Tableau / Power BI — visualisation and insights

## Goal
To determine whether entering the Indian creator gear market 
is financially viable, and to define the best entry strategy 
for a new brand.
